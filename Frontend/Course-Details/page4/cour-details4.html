Overview of Reconnaissance and Footprinting

This module introduces the initial phases of ethical hacking: reconnaissance and footprinting. These are essential steps where information about a target is gathered without directly engaging with the system.

Learning Objectives

  • Understand the importance of reconnaissance in ethical hacking.
  • Learn the different types of reconnaissance.
  • Explore footprinting techniques and tools used to gather information.

Types of Reconnaissance

Reconnaissance can be categorized into two main types:

  • Passive Reconnaissance: Gathering information without direct interaction with the target. This might involve using public sources, like search engines and social media.
  • Active Reconnaissance: Involves direct interaction with the target to gather information. This might include using scanning tools to check for open ports or services.

Footprinting Techniques

Footprinting techniques are used to gather information such as domain names, IP addresses, and other network details. Common methods include:

  • WHOIS Lookups: To obtain registration details about domain names.
  • DNS Enumeration: Gathering information about the target's domain and subdomains.
  • Social Engineering: Using psychological manipulation to obtain sensitive information.
  • Search Engines: Leveraging search engines to gather public information about the target.

Popular Tools for Reconnaissance and Footprinting

  • Maltego: A tool for data mining and gathering information about individuals, companies, or networks.
  • Shodan: A search engine for Internet-connected devices that helps find specific IPs, open ports, and exposed services.
  • Google Dorks: A search technique to use Google to reveal specific types of information.
  • Nslookup: A command-line tool for querying DNS to obtain domain and IP address information.
